# bootcamp-dev

[Boot Camp API](https://bootcamp-dev.azurewebsites.net)

## Getting Start

  ```
git clone https://github.com/boybundit/bootcamp-dev.git
cd bootcamp-dev
npm install
npm start
  ```

## Test
mocha
  ```
npm test
  ```
  
## Documentation

  ```
npm run apidoc
  ```
