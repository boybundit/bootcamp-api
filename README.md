# Boot Camp API

[Boot Camp API](https://bootcamp-dev.azurewebsites.net)

## Getting Start

  ```
git clone https://github.com/boybundit/bootcamp-dev.git
cd bootcamp-dev
npm install
set DB_USERNAME={{USERNAME}}
set DB_PASSWORD={{PASSWORD}}
node src/server.js
  ```

## Test

To run mocha test

```
npm test
  ```
  
## Documentation

To generate API documentation using apidoc

  ```
npm run apidoc
  ```
